# About
This API has goal to manage allocation of professors in courses at the educational institution.

## Diagram class
![imagem1](https://github.com/abnerjosefelixbarbosa/assents/blob/main/modelo_professor_allocation.png)

# Features

- find all allocations
- find a allocation
- save a allocation
- update a allocation
- delete a allocation
- delete all allocations

- find all courses
- find a course
- save a course
- update a course
- delete a course
- delete all courses

- find all departments
- find departments by name
- find a department
- save a department
- update a department
- delete a department
- delete all departments

- find all professors
- find a professor
- save a professor
- update a professor
- delete a professor
- delete all professors
